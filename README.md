# 🔐 Password Generator

A modern Password Generator built using React, Vite, and Tailwind CSS. It allows users to generate secure passwords with customizable length, numbers, and special characters.

## 🚀 Features

* Generate secure random passwords
* Customize password length
* Include numbers
* Include special characters
* Copy password to clipboard with one click
* Responsive and user-friendly interface
